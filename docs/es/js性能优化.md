

##  JS性能优化

### 什么是性能优化

所谓的性能优化就是提升代码的运行效率、减少代码运行时资源消耗。而JS性能优化就是在这门语言中优化代码的性能。

### JS的内存管理

内存:可读写单元，表示一片可操作的区域。

管理:人为的操作一片空间的申请、使用和释放

内存管理：开发者主动去申请内存空间、使用内存空间、和释放内存空间

JS不能想C语言那样主动去申请和释放内存空间，但是我们我还能还是可通过执行脚本来观察JS中内存使用的生命周期变化

```js
/**
 *
 * 内存管理
 *
 */

// 申请内存

let a = {}

// 使用内存

a.name = 'tom'

// 释放内存

a = null

```
