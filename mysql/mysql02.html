    # str_to_date()

    STR_TO_DATE(str,format)函数是将时间格式的字符串（str），按照所提供的显示格式（format）转换为 DATETIME 类型的值。

    SELECT STR_TO_DATE(now(),'%Y-%m-%d %H:%i:%s') AS result;

SELECT * from employees where hiredate=str_to_date('1992/4/3','%Y/%m/%d')
